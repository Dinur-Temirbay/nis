# НИШ — Landing Page 🎓

Информативный лендинг для образовательного центра AiPLUS, посвященный подготовке школьников к поступлению в НИШ. Проект демонстрирует навыки работы с многослойным дизайном, кастомными формами и базовой адаптивностью.

## 🛠 Технологический стек

- **HTML5:** Семантическая верстка, использование форм обратной связи и интеграция внешних иконок.
- **CSS3:** - Продвинутое позиционирование слоев, сетки на Flexbox и кастомная стилизация элементов.
- **Библиотеки:** - [Font Awesome 6](https://fontawesome.com/) — использование векторных иконок для соцсетей и интерфейса.
- **Fonts:** Локальное подключение шрифтов через `@font-face` (Gilroy, HeadingPro).

---

## 🚀 Как запустить проект

Для корректной работы всех функций и правильного отображения графических ресурсов рекомендуется использовать локальный сервер.

### Использование Live Server (рекомендуется):
1. Откройте папку проекта в **VS Code**.
2. Установите расширение **Live Server** (если еще не установлено).
3. Нажмите на кнопку **Go Live** в статус-баре (внизу справа) или кликните правой кнопкой мыши по `index.html` и выберите **"Open with Live Server"**.
4. Проект автоматически откроется в браузере по адресу `http://127.0.0.1:5500`.
